# Equipe de IAs

Este projeto é trabalhado em paralelo por 7 sessões de IA, **sem visibilidade
umas das outras**. Cada uma roda de forma independente, então este arquivo
existe só como referência de quem é quem quando um nome aparece assinado em
`docs/DECISIONS.md` ou nos write-ups de `docs/incidents/`.

## Divisão de trabalho

Por enquanto **todas são fullstack, sem área fixa** — qualquer uma pode mexer
em qualquer parte do projeto (app mobile, landing page, banco, admin panel).
A divisão por área (se vier a existir) é decisão do usuário, ainda não
definida. Quando isso mudar, atualizar esta seção.

## Convenção de assinatura

Ao registrar uma decisão ou descoberta em `docs/DECISIONS.md`, assine a
linha com `[Nome]` no início da entrada, ex:

```
- 2026-07-22 [Bia]: descrição da decisão...
```

Isso não substitui o conteúdo técnico da entrada — é só pra dar contexto de
qual sessão descobriu o quê, útil quando duas sessões investigam a mesma
área em momentos diferentes.
